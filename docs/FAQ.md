# Frequently Asked Questions

## General

### What is Comply Vault?

Comply Vault is a compliance documentation tool that converts Zoom meeting recordings into exam-ready client interaction records. Upload a recording, review extracted compliance data, and export an audit pack.

### Who is this for?

Comply Vault is designed for Registered Investment Advisors (RIAs) in the United States, particularly firms with 1-50 advisors.

### How does it work?

1. Upload a meeting recording
2. System transcribes and extracts compliance-relevant information
3. You review and edit the extracted data
4. Finalize the record
5. Export an audit pack (PDF, CSV, TXT, ZIP)

## Security & Compliance

### Is my data secure?

Yes. All data is encrypted in transit (TLS 1.2+) and at rest (AES-256). Workspace isolation ensures no cross-tenant access.

### How long is data retained?

Minimum 5 years (SEC requirement), default 6 years. Configurable per workspace.

### Can I delete data?

Yes, workspace owners can request data deletion. Legal hold prevents deletion when active.

### Is this SEC-compliant?

Comply Vault generates exam-ready audit packs that meet SEC Rule 204-2 requirements. You review and approve all records before filing.

## Troubleshooting

### Upload stuck on "Uploading"

- Check your internet connection
- Try refreshing the page
- Verify file size is under 500MB
- Try uploading again

### Processing stuck on "Processing"

- Click "Retry Processing" button
- Check if transcription/extraction services are available
- Contact support if issue persists

### Can't access a meeting

- Verify you're in the correct workspace
- Check if you have the right role (Member or Owner/CCO)
- Contact workspace owner if you need access

### Export fails

- Try again (may be temporary)
- Check if meeting is finalized or draft ready
- Contact support if issue persists
