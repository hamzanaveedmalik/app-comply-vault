# Comply Vault - User Guide

## Getting Started

### Creating Your Workspace

1. Sign in with your Discord account
2. Create a new workspace for your firm
3. Enter your workspace name (e.g., "Smith Financial Advisors")
4. Optionally enter a pilot code if you have one
5. Complete the $500 setup fee (or use pilot code for free setup)
6. Your 60-day free pilot period begins automatically

### Inviting Team Members

1. Go to Settings â†’ Invite Users
2. Choose "Single Invite" or "Bulk Invite"
3. Enter email addresses and assign roles:
   - **Owner/CCO:** Can finalize records and manage settings
   - **Member:** Can view and edit meetings
4. Team members receive invitation emails
5. They accept invitations and join your workspace

## Uploading Meetings

### Upload Process

1. Click "Upload Meeting" from the dashboard
2. Fill in meeting details:
   - Client Name
   - Meeting Type (e.g., "Portfolio Review")
   - Meeting Date
3. Confirm you have permission to upload the recording
4. Select your audio/video file (MP3, MP4, WAV, M4A, max 500MB)
5. Click "Upload"
6. File uploads directly to secure storage
7. Processing begins automatically

### Processing Status

- **Uploading:** File is being uploaded
- **Processing:** Transcription and extraction in progress
- **Draft Ready:** Ready for your review
- **Draft:** You've started editing
- **Finalized:** Ready for export (read-only)

## Reviewing and Editing

### Review Interface

1. Navigate to a meeting from the dashboard
2. View the two-column layout:
   - **Left:** Transcript with timestamps
   - **Right:** Extracted fields (topics, recommendations, disclosures, decisions, follow-ups)
3. Click evidence links to jump to transcript segments
4. Review soft gap prompts (warnings for missing disclosures)

### Editing Fields

1. Click "Edit" on any field section
2. Add, edit, or remove items:
   - **Text:** The content
   - **Reason:** Why you're making this change (optional)
   - **Timestamp:** Click transcript segments to auto-fill
3. Changes save automatically
4. Compliance note regenerates instantly

### Version History

- View all changes made to the meeting
- See who made each change and when
- Review edit reasons
- Track complete audit trail

## Finalizing Records

### Finalization Process

1. Review all extracted fields
2. Make any necessary edits
3. Mark as "Ready for CCO" (optional workflow step)
4. Click "Finalize Meeting" (Owner/CCO only)
5. Confirm finalization
6. Record becomes read-only
7. Time-to-Finalize is calculated and tracked

### After Finalization

- Record cannot be edited
- Export audit pack available
- Record appears in interaction log
- Included in finalization metrics

## Exporting Audit Packs

### Export Process

1. Navigate to a finalized (or draft ready) meeting
2. Click "Export Audit Pack"
3. System generates:
   - **PDF:** Structured compliance note
   - **CSV:** Evidence map
   - **CSV:** Version history
   - **TXT:** Full transcript
   - **CSV:** Interaction log entry
4. All files bundled in ZIP
5. Download starts automatically

### Export Contents

- **Compliance Note (PDF):** Meeting details, extracted fields, sign-off
- **Evidence Map (CSV):** All claims linked to timestamps and transcript snippets
- **Version History (CSV):** Complete edit history
- **Transcript (TXT):** Full transcript with timestamps
- **Interaction Log (CSV):** Summary entry for compliance systems

## Search and Discovery

### Global Search

1. Press `Cmd/Ctrl + K` or click search bar
2. Search by:
   - **Client Name:** Type client name
   - **Date Range:** "2024-01-01 to 2024-12-31"
   - **Keywords:** Search across transcripts and fields
3. View results with snippets
4. Click to navigate to meeting

### Interaction Log

1. Navigate to "Interaction Log"
2. View all meetings in table format
3. Filter by:
   - Client name
   - Date range
   - Meeting type
   - Recommendations (Y/N)
   - Finalized (Y/N)
4. Sort by any column
5. Export filtered results

## Review Queue

### Using the Review Queue

1. Navigate to "Review Queue"
2. View all meetings ready for review (Draft Ready or Draft)
3. Filter by:
   - Client name
   - Status
   - Date range
4. See "Ready for CCO" indicator
5. Click to review and finalize

## Settings

### Workspace Settings

1. Go to Settings
2. Configure:
   - **Retention Years:** Minimum 5, default 6
   - **Legal Hold:** Prevent deletion when active
3. Update billing information
4. Manage team members

## Troubleshooting

### Upload Issues

- **File too large:** Maximum 500MB, compress or use smaller file
- **Invalid format:** Use MP3, MP4, WAV, or M4A
- **Upload fails:** Check internet connection, try again
- **Stuck on processing:** Click "Retry Processing" button

### Processing Issues

- **Transcription fails:** File may be corrupted, try re-uploading
- **Extraction fails:** Transcript may be too short, try reprocessing
- **Processing stuck:** Use retry button or contact support

### General Issues

- **Can't access meeting:** Verify you're in the correct workspace
- **Can't finalize:** Only Owner/CCO can finalize
- **Export fails:** Try again, contact support if persists

## Support

- **Response Time:** Within 24 hours
- **Emergency:** For critical issues, include "URGENT" in subject

## Best Practices

1. **Upload promptly:** Upload recordings within 24 hours of meeting
2. **Review thoroughly:** Check all extracted fields before finalizing
3. **Add timestamps:** Link evidence to transcript timestamps
4. **Document edits:** Include reasons when editing fields
5. **Export regularly:** Download audit packs for your records
6. **Monitor metrics:** Track Time-to-Finalize on dashboard
